氢氧化镍真的和黑魂差很大

Icereal 3月 13, 2017
  • Games
  • 评测
  •  浏览量
  • 分享到 Twitter
  • 分享到 Google+
  • 分享到微博

在玩过《仁王》a、b测之后，见到IGN给出的最终评分9.6，我还是很惊讶的，毕竟血源也只拿到了9.1分。

《仁王》最初是在2004年开始制作的游戏（没能查清当初的游戏定位），起初，是以PS3护航作的身份亮相，然而之后10年消无声息，知道14年才再次有消息放出，将发售时间定在16年（后推迟至17年初）。时间跨度如此长的游戏，的确也是难得一见。

游戏形式

a测

最初的a测中，我们所见的游戏形式和魂系列（Dark Souls）游戏如出一辙。然而在玩家经历了黑暗之魂3的低难度代入和血源诅咒之后，“受死游戏”几乎不能有所突破（期待魂3第二个dlc）。而a测中的NioH俨然就是另一个受死游戏，甚至让玩家在游戏初期体会到了奇高的难度（个人感觉难过了亚南中心），尤其是精力系统，精力耗尽的惩罚极高，而pt人形小怪却很难耗尽精力。种种导致当初玩家（我）对这个开发了10+年的游戏并不看好。

b测

在b测中，动作系统进行了丰富，添加了部分武器闪避残心（不确定，不记得了），使得在手感上的难度有所降低，然而仍然没能让人感受到真正区别于魂系列的地方，有种另一个堕落之王的感觉。

正式版本

正式版发售之后，各大媒体评分却不约而同的出乎意料，而玩家中的口碑也迅速累积，这让人很惊讶。在正式版中，丰富的动作系统有效的降低了游戏难度，并且AI设计上也与a测有明显的难度降低，而且丰富的动作系统使得游戏前中期的打法显得丰富（然而极不平衡的部分技能也使得打法趋同“双刀水形剑”），武器系统使得游戏后期难度趋于平缓，整个一周目并没有类似黑魂血源中可能出现的“受苦”一下午一晚上的特定地点或是特定BOSS存在。而仁王的任务系统则和“受死游戏”有很大不同，仁王将任务设置成地图性的，即每次所进的地图只有一个任务，并且任务明确，并没有隐晦的叙述或者需要玩家仔细研究的部分。这种方案的地图的确使得玩家的负担（记忆负担）小了很多，同时让暗荣地图设计上的不足藏了起来。

一点想法

总的来说，仁王算得上是当前的一个很成功的“受死游戏”，同时他也摆脱了魂系列的影子：完全不同的动作系统、分隔开的地图、突出历史感的剧情（虽然经不起推敲）（然而历史的厚重感渲染不如Ubi），同时能将具有忍龙风格的动作系统和受死结合在一起，使得受死游戏和动作游戏门槛同时降低，的确是很优秀的设计，再加上西方人文化的冲突（威廉酒席上的不适）以及最终BOSS的血源串场感，可以说是“还行”了。

然而，游戏内部部分系统的严重不平衡，使得游戏体验趋于单调。双刀的水形剑，无限居合和十字斩，忍术部分线伤害严重不足（伊贺流不如埋火），看奖杯就知道斧几乎没人用。到了游戏后期，大量的技能显得多余，玩家获得了足够好的装备，只需要躲避绕后几刀就足以应对大部分小怪，使得二周目的高难度没能体现出该有的水平。

IGN这个9.6分，总还是给高了。
